# Data Science Projects

My data-science projects, one folder per project. Each is self-contained with its
own data, notebook, figures and README.

## Projects

| Project | Description |
|---------|-------------|
| [ai-energy-consumption](ai-energy-consumption/) | Estimates vs measured energy cost of an AI query. Widely-cited 2024 numbers run several times higher than recent production measurements; long-reasoning models are the real outliers. |

## Layout

```
Data-Science-Projects/
├── README.md
├── .gitignore
└── <project>/
    ├── README.md
    ├── <notebook>.ipynb
    ├── data/
    └── figures/
```

Notebooks are committed with their outputs so the analysis is viewable directly on
GitHub.
